Questions

  1. The server in the client server architecture is software running on a computer. The server is responding to requests from the client.
  2. The client is the browser. It locate and display information from the web.
  3. IP stands for "Internet Protocol". An IP address is a unique address that every device on the Internet are given so that they can communicate across the Internet. Each IP address has four parts, each part a number between 0 and 255.
  4. DNS stands for "Domain Name System" and it's the phonebook of the Internet. Because it would be difficult to remember an IP address when you are entering a website, every website has its own domain name, like facebook.com. The task of the DNS is to translate this domain into an IP address so that the browser can load Internet resources.
  5. The difference between the Internet and the WWW is that WWW are the pages you see when you are at a device and you are online, like for example facebook.com. But the Internet is the network of connected computers that the web works on. This is not someting you can see, like a webpage.
  6. HTML stands for "Hyper Text Markup Language" and is the standard coding language for creating webpages. The language describes the structure of a webpage and consists of a series of elements.
